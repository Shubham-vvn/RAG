# 🖥️ Ask My Documents — Frontend Application

This is the Next.js frontend application for the RAG-grounded document search application.

## 📖 Main Documentation
For complete information regarding the architecture, backend setup, local embeddings model details, and RAG pipeline configurations, please refer to the primary project **[README.md](../README.md)** at the root level of the workspace.

## 🚀 Quick Start
To launch the Next.js development server locally:

```bash
# 1. Install dependencies
npm install

# 2. Run the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the application interface.
